# Live2D Models

为 [l2d](https://github.com/hacxy/l2d) 提供 Live2D 模型资源的 CDN 仓库。

> [oh-my-live2d](https://github.com/oh-my-live2d/oh-my-live2d) 已废弃，请使用 [l2d](https://github.com/hacxy/l2d) 代替。

## 使用方式

所有模型通过 `https://model.oml2d.com` 访问，格式如下：

```
https://model.oml2d.com/{模型名称}/{模型json文件}
```

示例：

```
https://model.oml2d.com/Senko_Normals/senko.model3.json
```

## 预览

前往 [Live2D Viewer](https://l2d-viewer.hacxy.cn/) 在线预览和使用这些模型。
